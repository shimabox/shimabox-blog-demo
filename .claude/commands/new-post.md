# 新規記事作成

以下の手順で新規記事を作成してください：

1. 今日の日付を取得（YYYY-MM-DD形式）
2. 引数の日本語タイトルを英語のslugに変換（ケバブケース、小文字、簡潔に）
3. `content/posts/YYYY-MM-DD-{slug}.md` を作成
4. `content/images/YYYY/MM/` ディレクトリがなければ作成
5. タイトルから連想されるサムネイル画像生成用プロンプト（英語）を出力
6. 作成完了を報告

## slug変換ルール

- 日本語タイトルを簡潔な英語に翻訳
- ケバブケース（単語をハイフンで区切る）
- すべて小文字
- 記事の内容を表す2〜5単語程度

例：
- 「Claude Codeを使ってみた」→ `trying-claude-code`
- 「React 19の新機能まとめ」→ `react-19-new-features`
- 「ブログをWordPressから移行した話」→ `migrating-blog-from-wordpress`

## Markdownテンプレート

以下のプレースホルダーをすべて実際の値に置換して記事を作成：
- `{日本語タイトル}` → 引数で渡された日本語タイトル
- `{slug}` → 英語に変換したslug
- `{YYYY-MM-DD}` → 今日の日付（例: 2026-01-12）
- `{YYYY}` → 年（例: 2026）
- `{MM}` → 月（例: 01）

```markdown
---
title: "{日本語タイトル}"
slug: "{slug}"
date: "{YYYY-MM-DD}"
categories: []
image: "/images/{YYYY}/{MM}/{slug}.png"
---

ここに本文を書く
```

## 画像生成プロンプト

タイトルから連想されるサムネイル画像用のプロンプトを英語で生成する。

### プロンプトのルール

- DALL-E向けの英語プロンプト
- ブログのサムネイルとして適切なスタイル
- 写真風、イラスト風など記事の内容に合わせる
- シンプルで視認性が良いデザイン
- アスペクト比は16:9を想定

### 出力形式

```
🎨 画像生成プロンプト（ChatGPTにコピペ）:
{プロンプト}
```

## 引数

$ARGUMENTS があればそれを日本語タイトルとして使用。なければユーザーに聞く。
